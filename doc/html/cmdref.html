Het openen van een nieuw boekjaar kan te allen tijde worden uitgevoerd, uiteraard maar één keer per boekjaar. Het openen kan een aantal opdrachten omvatten, en wordt afgesloten met de opdracht "adm_open". Zolang deze opdracht niet is gegeven blijft de administratie ongewijzigd. Mogelijke opdrachten voor openen van een boekjaar: adm_naam "Naam van de administratie" adm_btwperiode [ jaar | kwartaal | maand ] adm_boekjaarcode Een code van max 4 letters en/of cijfers waarmee het boekjaar kan worden geïdentificeerd. Standaard wordt het jaartal van het boekjaar genomen. adm_open Alle informatie die met de bovenstaande opdrachten is ingevoerd, wordt verwerkt.
